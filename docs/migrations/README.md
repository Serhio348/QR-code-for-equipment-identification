# Миграции базы данных

Эта папка содержит SQL миграции для базы данных Supabase.

## Структура

- **add-device-passport-fields.sql** - Добавление полей паспорта счетчика (выполнено)
- **create-beliot-readings-table.sql** - Создание таблицы показаний Beliot (выполнено)

## Как использовать

1. Откройте Supabase Dashboard → ваш проект
2. Перейдите в SQL Editor
3. Скопируйте содержимое нужного файла миграции
4. Выполните скрипт

## Важно

- Миграции идемпотентны (можно выполнять многократно)
- Используйте `IF NOT EXISTS` и `ADD COLUMN IF NOT EXISTS` для безопасности
- После выполнения миграции можно оставить файл для истории
