# Справочные SQL запросы

Эта папка содержит примеры SQL запросов для работы с базой данных.

## Файлы

- **query-device-by-period.sql** - Запросы для получения данных по счетчику за период
- **query-past-hour.sql** - Запросы для получения данных за последний час

## Как использовать

1. Откройте Supabase Dashboard → ваш проект
2. Перейдите в SQL Editor
3. Скопируйте нужный запрос
4. Замените параметры (device_id, даты и т.д.) на свои значения
5. Выполните запрос

## Примечание

Эти запросы предназначены для справки и отладки. В приложении используются API функции.
