# Использование логотипа в приложении

## Текущая реализация

В header добавлен элемент `.header-logo`, который сейчас отображает текст "EQ" (Equipment).

## Как добавить свой логотип

### Вариант 1: SVG логотип (рекомендуется)

1. Поместите файл логотипа в `public/logo.svg`
2. Обновите `src/App.tsx`:

```tsx
<Link to={ROUTES.HOME} className="header-logo" title="Главное меню">
  <img src="/logo.svg" alt="Логотип" />
</Link>
```

### Вариант 2: PNG/JPG логотип

1. Поместите файл в `public/logo.png` (или `.jpg`)
2. Обновите `src/App.tsx` аналогично варианту 1

### Вариант 3: Иконка из папки icons

Если у вас уже есть иконки в `public/icons/`, можно использовать их:

```tsx
<Link to={ROUTES.HOME} className="header-logo" title="Главное меню">
  <img src="/icons/icon-192x192.png" alt="Логотип" />
</Link>
```

## Размеры логотипа

- **Десктоп**: 40x40px
- **Мобильные**: 36x36px

Логотип автоматически масштабируется благодаря CSS.

## Стилизация

Логотип имеет:
- Скругленные углы (10px)
- Градиентный фон (если нет изображения)
- Тень для глубины
- Hover эффект (можно добавить)

## Рекомендации

- Используйте SVG для лучшего качества на всех экранах
- Оптимизируйте размер файла (SVG обычно меньше)
- Убедитесь, что логотип читаем на светлом фоне
- Для темной темы может понадобиться отдельная версия
