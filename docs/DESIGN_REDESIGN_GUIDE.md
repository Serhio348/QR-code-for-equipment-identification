# Руководство по редизайну приложения

## Текущая цветовая схема

- **Основной градиент**: `#667eea` → `#764ba2` (фиолетово-синий)
- **Фон приложения**: `#f5f5f5` (светло-серый)
- **Карточки**: белый фон

## Процесс редизайна

### 1. Выбор цветовой схемы

Создан файл `src/styles/colors.css` с несколькими вариантами:

- **default** (текущая) - Фиолетово-синий градиент
- **ocean** - Синий океан
- **forest** - Зеленый лес
- **sunset** - Оранжевый закат
- **dark** - Темная тема
- **minimal** - Минималистичная светлая

### 2. Применение темы

Чтобы применить тему, добавьте атрибут `data-theme` к корневому элементу:

```tsx
// В App.tsx
<div className="app" data-theme="ocean">
```

### 3. Использование CSS-переменных

Замените хардкодные цвета на CSS-переменные:

**Было:**
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**Стало:**
```css
background: var(--gradient-primary);
```

### 4. Обновление компонентов

Основные файлы для обновления:

1. **Шапка приложения**: `src/App.css`
   - `.app-header` - фон шапки
   - `.header-title` - заголовок
   - `.nav-link` - навигационные ссылки
   - `.logout-button` - кнопка выхода

2. **Главное меню**: `src/pages/MainMenuPage.css`
   - `.main-menu-page` - фон страницы
   - `.main-menu-header` - заголовок меню
   - `.main-menu-button` - кнопки меню

3. **Страницы входа**: `src/pages/LoginPage.css`
   - `.login-page` - фон страницы
   - `.submit-button` - кнопка отправки

### 5. Примеры обновления

#### Шапка приложения

```css
.app-header {
  background: var(--gradient-primary);
  color: var(--text-inverse);
  box-shadow: var(--shadow-md);
}

.nav-link:hover {
  background: var(--hover-overlay);
}
```

#### Кнопки

```css
.submit-button {
  background: var(--gradient-primary);
  color: var(--text-inverse);
  box-shadow: var(--shadow-primary);
}

.submit-button:hover {
  background: var(--gradient-primary-hover);
}
```

### 6. Тестирование

После применения новой темы проверьте:

- ✅ Контрастность текста (доступность)
- ✅ Читаемость на всех экранах
- ✅ Соответствие бренду/стилю
- ✅ Единообразие во всех компонентах

### 7. Полезные ресурсы

- **Coolors.co** - генератор цветовых палитр
- **Adobe Color** - подбор гармоничных цветов
- **Material Design Colors** - готовые палитры
- **WebAIM Contrast Checker** - проверка контрастности

## Рекомендации

1. **Начните с шапки** - это первое, что видит пользователь
2. **Используйте CSS-переменные** - упростит будущие изменения
3. **Тестируйте на разных устройствах** - цвета могут выглядеть по-разному
4. **Сохраняйте контрастность** - минимум 4.5:1 для обычного текста
5. **Документируйте изменения** - для будущих обновлений
